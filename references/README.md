# Visual reference

`qr_reference.png` is a real QR Code downloaded for visual comparison from:

<https://api.qrserver.com/v1/create-qr-code/?size=1000x1000&ecc=H&margin=40&data=GDC%20QR%20visual%20reference>

It is used only as a design reference. GDC v10 generates its carrier matrix
locally with the `qrcode` package and does not require network access.
